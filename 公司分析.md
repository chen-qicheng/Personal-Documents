   ## 一、持有意愿判断
   1. 选项：A 愿意 / B 不愿意 / C 仍需观望
   2. 核心内容：
   - 核心理由：优点？、护城河？、长期现金流？等
   - 核心疑问：（如：第二曲线增长是否具备确定性）
   - 核心依赖：长期持有的理由及成立前提
   - 观察变量：监控依赖假设前提的变量
   - 商业模式本质：公司是怎样赚钱的？这套逻辑换做其他公司是否还能成立？

   ## 二、收入与利润分析
   1. 收入：各收入来源及占比，
   2. 成本：a/包括收入成本、销售推广成本、前程开支，研发支出等，b/未来成本大幅上升可能性
   3. 利润驱动因素：包括毛利率、成本效益优化、投资收益
   4. 资源与成本的优化空间
   5. 规模效应、网络效应、生态闭环效应等
   6. 商业模式中最脆弱的一环?

   ## 三、行业与竞争分析
   1. 行业发展阶段：A 成熟 / B 成长期/ 其他，明确公司的主要领域和细分领域
   2. 监管与政策影响：分业务、分板块梳理
   3. 主要竞争对手：分业务、分板块梳理
   4. 公司在行业中的相对位置：分业务、分板块梳理
   5. 行业恶化情景下，公司是否比同行活得更久
   6. 公司是否存在不可替代性

   ## 四、管理层资本配置
   1. 管理层是否坚持长期主义
   2. 沟通与信息透明度
   3. 资本配置优先级：现金储备、研发投入、股份回购、分红等
   4. 管理层是否像股东一样思考

   ## 五、财务质量分析
   1. 账务质量和盈利质量：包括ROE、现金流等
   2. 杠杆与财务风险
   3. 一句话财务总结
