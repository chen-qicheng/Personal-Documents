### 【问题分类】结构设计-尺寸冲突 ✅ 已修复
### 【问题描述】扬声器安装空间存在严重尺寸冲突
### 【位置定位】1.3-结构设计说明书.md 第4.2.5节、第5.3节
### 【修复方案】综合优化方案（A+B+C）

**原问题分析**：
- 扬声器规格：Φ40×18mm（高度18mm）
- 卡槽深度设计：5mm
- 上盖内部空腔：仅8mm
- **冲突**：18mm扬声器无法放入5mm+8mm=13mm的总空间

**修复内容**：
1. **下盖卡槽深度**：5mm → **8mm**（增加3mm）
2. **上盖出声通道**：8mm → **10mm**（增加2mm）
3. **整体外形高度**：40mm → **42mm**（增加2mm）
4. **内部净空间高度**：28mm → **30mm**（增加2mm）
5. **模块堆叠优化**：
   - 原布局：主板顶部到模块7mm，模块到扬声器8mm
   - 新布局：主板顶部到模块5mm，模块到扬声器5mm

**修复后空间分配**：
```
总高度42mm = 底壳2mm + 电池18mm + 主板10mm + 模块5mm + 扬声器间隙5mm + 上盖12mm

扬声器安装空间 = 下盖卡槽8mm + 上盖空腔10mm = 18mm ✓
```

**验证**：18mm扬声器完美适配18mm总安装空间，无干涉。

### 【问题分类】结构设计-模块安装空间 ✅ 已修复
### 【问题描述】HC-SR501红外模块安装空间不足
### 【位置定位】1.3-结构设计说明书.md 第2.1节、第4.1.3节
### 【修复方案】下壳倾斜支架设计

**原问题分析**：
- HC-SR501模块高度：25mm
- 上盖总高度：仅12mm
- **冲突**：25mm模块无法装入12mm高的上盖空间

**修复内容**：
1. **安装位置调整**：上盖内嵌式 → **下壳前部倾斜支架**
2. **新增倾斜支架设计**（第4.2.6节）：
   - 支架尺寸：32×24×8mm底座，与下壳一体打印
   - 倾斜角度：15°，使模块感应方向斜向上对准窗口
   - 固定方式：双面胶/热熔胶固定模块
3. **红外窗口重新设计**：
   - 尺寸：18×14mm → **20×18mm**（增大以适配倾斜视角）
   - 倾斜角度：**15°向上倾斜**
   - 安装方式：内嵌式 → **外嵌式**（透过窗口直接看到模块透镜）
4. **布局图更新**：在下壳布局中标注红外模块位置（电池右侧，15°倾斜）

**修复后空间分配**：
```
红外模块总高度25mm = 支架底座8mm + 模块高度25mm（部分在下壳空腔）
                    ↓
        模块向上倾斜15°，透镜对准上盖窗口
        感应方向：斜向上，适合柜台检测前方来人
```

**附加优势**：
- 简化装配：无需单独安装菲涅尔透镜片
- 优化感应：15°倾斜角更适合柜台高度检测消费者
- 便于维护：模块在下壳，拆卸上盖即可检修
