## 1. 概述

### 1.1 设计目标

本文档详细描述新品卷烟推广语音播报器的结构设计方案，包括外壳造型、内部布局、接口开孔、固定方式等，为3D打印制作提供完整的技术依据。

### 1.2 设计原则

| 原则 | 说明 |
|------|------|
| 美观性 | 外观简洁大方，符合卷烟柜台展示需求 |
| 功能性 | 满足所有硬件模块的安装和功能实现 |
| 可制造性 | 适合FDM 3D打印（PLA耗材），支撑易去除 |
| 可装配性 | 结构简单，便于手工组装和调试 |
| 成本可控 | 单件打印成本控制在50元以内 |

### 1.3 设计依据

- 《0-需求分析.md》外观和安装要求
- 《1.1-硬件详细设计说明书.md》模块尺寸和接口定义
- 卷烟柜台实际使用场景调研

## 3. 整体结构方案

### 3.1 结构选型

采用 **上下分体式外壳** 设计：

```
┌─────────────────────────────────────────┐
│              上盖 (Top Cover)            │
│         ┌───────────────────┐           │
│         │   红外窗口/LED    │           │
│         ├───────────────────┤           │
│         │                   │           │
│         │    扬声器格栅      │           │
│         │                   │           │
│         └───────────────────┘           │
├─────────────────────────────────────────┤
│              下壳 (Main Body)            │
│    ┌─────────────────────────────────┐  │
│    │                                 │  │
│    │         内部安装空间             │  │
│    │      (主板/电池/模块)           │  │
│    │                                 │  │
│    └─────────────────────────────────┘  │
│         │ USB │开关│Type-C│            │
└─────────────────────────────────────────┘
```

**选型理由**:
- 上下分体便于内部模块安装和调试
- 上盖可设计美观的出声孔和窗口
- 下壳集中所有接口，便于操作
- 适合FDM打印，PLA耗材成本低廉，支撑易去除

**设计要点**：

3. **感应范围**：水平100°视角，探测距离1-3米
4. **支架强度**：与下壳一体打印，壁厚2mm，底部加强筋

## 5. 内部模块布局

### 5.1 布局原则

1. **重心稳定**: 电池和扬声器靠近底部
2. **散热考虑**: 功放远离红外传感器
3. **信号干扰**: 音频模块远离数字高频部分
4. **维护便利**: 电池可更换，调试接口可达

## 8. 材料与工艺

### 8.1 材料选择

**推荐材料**: PLA Basic（聚乳酸基础款）

| 材料类型 | 特点 | 适用性 |
|----------|------|--------|
| PLA Basic | 成本低、易打印、环保无毒、表面质感好 | ★★★ 推荐 |
| PLA+ / PLA Pro | 强度更高、韧性更好 | ★★☆ 可选 |
| PETG | 耐热性好、耐冲击 | ★★☆ 高温环境可选 |
| ABS | 强度高、可抛光 | ★☆☆ 需封箱打印 |

**PLA Basic 材料参数**: 
- 密度: 1.24 g/cm³
- 抗拉强度: 50-60 MPa
- 热变形温度: 55-60°C
- 打印温度: 200-220°C
- 平台温度: 50-60°C

### 8.2 打印工艺参数

| 参数 | 设置值 |
|------|--------|
| 层厚 | 0.2mm (标准) / 0.12mm (精细) |
| 壁厚 | 3层 (约1.2mm) |
| 填充密度 | 15-20% (外壳) / 30% (安装位) |
| 支撑方式 | 树状支撑 / 普通支撑，悬空角>45° |
| 摆放方向 | 最大面朝下，减少支撑 |
| 打印速度 | 50mm/s (外墙) / 80mm/s (内填充) |

### 8.3 打印摆放建议

```
上盖打印摆放 (推荐)

    ┌───────────────────┐
    │                   │
    │    内侧朝上       │  ◄── 打印平台
    │    (支撑少)       │
    │                   │
    │  ┌─────────────┐  │
    │  │ 扬声器区域   │  │
    │  │  (向下悬空)  │  │  ◄── 需要树状支撑
    │  └─────────────┘  │
    └───────────────────┘
    
下壳打印摆放

    ┌───────────────────┐
    │                   │
    │    外侧朝上       │  ◄── 打印平台
    │    (开口朝上)     │
    │                   │
    │  内部结构朝上     │
    │  (需要少量支撑)   │
    └───────────────────┘

PLA打印优势：
- 成本低廉：相比光敏树脂成本降低60%以上
- 打印便捷：FDM打印技术成熟，支撑易去除
- 环保安全：PLA为生物降解材料，无毒无味
- 后期处理：可打磨、喷漆、粘合
```
