## 1. 概述

### 1.1 测试目的

验证新品卷烟推广语音播报器满足需求规格说明书中定义的全部功能、性能和质量要求，确保设备在卷烟柜台场景下稳定可靠运行。

### 1.2 测试范围

| 测试类型 | 测试对象 | 测试阶段 |
|----------|----------|----------|
| 硬件测试 | 各硬件模块及接口 | 硬件调试阶段 |
| 软件测试 | 固件功能及逻辑 | 软件开发阶段 |
| 集成测试 | 软硬件联调 | 联调测试阶段 |
| 性能测试 | 功耗、续航、响应 | 集成测试阶段 |
| 可靠性测试 | 稳定性、环境适应 | 验收前测试 |
| 结构测试 | 3D打印件装配 | 结构制作阶段 |
| 验收测试 | 全功能场景测试 | 项目收尾阶段 |

### 1.3 测试环境

**硬件环境**:
- 被测设备: 语音播报器样品
- 测试仪器: 万用表、示波器、可调电源、电子负载
- 辅助设备: USB测试U盘、标准音频文件

**软件环境**:
- 调试工具: Keil MDK / STM32CubeIDE
- 串口工具: SSCOM / XCOM
- 音频文件: 标准MP3 (128kbps, 44.1kHz)

**环境条件**:
- 温度: 15℃~35℃
- 湿度: 45%~75%RH
- 电源: 5V±0.25V (Type-C供电)

## 3. 硬件测试

### 3.1 电源模块测试

#### TC-PWR-001 电池电压检测

| 项目 | 内容 |
|------|------|
| **测试目的** | 验证电池电压检测电路准确性 |
| **测试条件** | 可调电源模拟电池，电压范围3.0V-4.2V |
| **测试步骤** | 1. 连接可调电源至电池接口<br>2. 设置电源电压3.0V，读取ADC值<br>3. 以0.2V步进增加至4.2V，记录各点<br>4. 对比实际值与测量值 |
| **预期结果** | 电压测量误差≤±0.1V，线性度良好 |
| **测试等级** | P0（关键） |

#### TC-PWR-002 充电功能测试

| 项目 | 内容 |
|------|------|
| **测试目的** | 验证TP4056充电功能及状态指示 |
| **测试条件** | 电池电压<4.0V，Type-C接入5V电源 |
| **测试步骤** | 1. 接入 discharged 电池（约3.5V）<br>2. 插入Type-C充电线<br>3. 观察充电指示灯（应红灯亮）<br>4. 等待充电完成，观察指示灯（应绿灯亮）<br>5. 测量充满电压（应为4.2V±0.05V） |
| **预期结果** | 正常充电，指示灯状态正确，充满电压4.2V |
| **测试等级** | P0（关键） |

#### TC-PWR-003 升压输出测试

| 项目 | 内容 |
|------|------|
| **测试目的** | 验证5V升压输出稳定性 |
| **测试条件** | 电池电压3.0V-4.2V变化，负载0-200mA |
| **测试步骤** | 1. 设置电池电压3.0V<br>2. 测量5V输出电压<br>3. 增加负载至200mA<br>4. 记录各负载点输出电压<br>5. 重复测试电池电压3.7V、4.2V |
| **预期结果** | 输出电压5.0V±0.25V，负载调整率<5% |
| **测试等级** | P0（关键） |

#### TC-PWR-004 低功耗休眠电流

|------|------|
